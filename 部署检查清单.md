# ✅ 部署检查清单

在部署到Streamlit Cloud之前，请确保完成以下检查：

## 📋 代码准备

- [ ] `requirements.txt` 包含所有依赖（包括streamlit）
- [ ] `app.py` 可以正常运行
- [ ] `.gitignore` 已配置（避免提交敏感文件）
- [ ] 代码已推送到GitHub仓库

## 🔐 环境变量

- [ ] 准备API密钥（DASHSCOPE_API_KEY）
- [ ] 在Streamlit Cloud的Secrets中配置环境变量
- [ ] 确保不会在代码中硬编码API密钥

## 📁 数据文件

- [ ] 测试数据已包含在仓库中（或使用云端存储）
- [ ] 数据文件大小合理（<100MB，或使用Git LFS）
- [ ] 向量数据库文件已预生成（如果需要）

## 📝 文档完善

- [ ] README_CN.md 已完善
- [ ] 项目描述清晰
- [ ] 使用说明完整

## 🚀 部署步骤

1. [ ] 访问 https://share.streamlit.io/
2. [ ] 使用GitHub账号登录
3. [ ] 点击 "New app"
4. [ ] 选择GitHub仓库
5. [ ] 设置 Main file path: `app.py`
6. [ ] 配置Secrets（环境变量）
7. [ ] 点击 Deploy
8. [ ] 等待部署完成
9. [ ] 获取公开链接
10. [ ] 测试功能是否正常

## ✨ 部署后

- [ ] 测试基本功能
- [ ] 更新简历中的链接
- [ ] 准备项目演示说明
- [ ] 录制演示视频（可选）
